## üì∑ Camera Settings

### Beach (Bright Sun)
```
Mode: Manual
ISO: 100-400
Aperture: f/8-f/11
Shutter: 1/125 - 1/500
White Balance: Daylight (5500K)
Format: RAW + JPEG
```

### Rooms (Mixed Light)
```
ISO: 400-800
Aperture: f/5.6-f/8
Shutter: 1/60 - 1/125
White Balance: Mixed
Lens: 24mm or 16-35mm
Tripod: Recommended
```

### Food (Natural Light)
```
ISO: 100-400
Aperture: f/2.8-f/5.6
Shutter: 1/125 - 1/250
White Balance: Daylight
Lens: 50mm or 100mm macro
Tripod: Recommended
```

### Evening/Night
```
ISO: 800-3200
Aperture: f/2.8-f/4
Shutter: 1/60 - 1/250
White Balance: Auto or Tungsten
Tripod: Required
```
